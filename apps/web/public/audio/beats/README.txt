Circuit Cypher rap beats (drop-in)

Put instrumental rap-beat mp3 files in this folder, then list their file
names in manifest.json, e.g.:

  { "tracks": ["prettyjohn1-trap.mp3", "gr0za-boombap.mp3", "beat-3.mp3"] }

The host loads them on start and cycles them per game (a seeded shuffle of
the room code), so the MC and each performer get a different song every
match. With an empty list (the default) Circuit Cypher uses its built-in
synthesized boom-bap beat instead, so nothing breaks when the folder is empty.

Suggested source: https://pixabay.com/music/search/rap%20beat/
(Pixabay tracks are free for commercial use, no attribution required.)
Download the tracks you like (the prettyjohn1 and gr0za ones were called out),
drop the mp3s here, and add their file names above.
